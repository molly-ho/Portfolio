portfolio

portfolio columns

Use .col-2, .col-3, .col-4 classes to set column numbers


<!-- PORTFOLIO-WRAPPER -->
<div class="portfolio-wrapper container">
	<!-- PORTFOLIO -->
	<div class="portfolio col-3">
		...
	</div>
	<!-- /PORTFOLIO -->
</div>
<!-- /PORTFOLIO-WRAPPER -->

portfolio item


<!-- PORTFOLIO ITEM -->
<div class="item">
	<a href="project-link.html" class="item-link">
		<div class="item-img" style="background-image: url('image-url.jpg')"></div>
		<div class="item-overlay">
			<h5>item title</h5>
		</div>
	</a>
</div>
<!-- /PORTFOLIO ITEM -->

filters

filter buttons

Add filters with data-group attribute


<div class="filter">
	<ul>
		<li data-group="all" class="active">all</li>
		<li data-group="photography">photography</li>
		<li data-group="branding">branding</li>
		<li data-group="design">design</li>
	</ul>
</div>

portfolio item category

For your portfolio items to assign category use data-groups attribute.


<!-- PORTFOLIO ITEM -->
<div class="item" data-groups="branding">
	...
</div>
<!-- /PORTFOLIO ITEM -->

You can assign multiple categories(comma separated without spaces)


<!-- PORTFOLIO ITEM -->
<div class="item" data-groups="branding,design">
	...
</div>
<!-- /PORTFOLIO ITEM -->

spacers & separators

margins

use .mrg-{10, 20, ... 120}, .mrg-top-{10, 20, ... 120}, .mrg-btm-{10, 20, ... 120} classes


<!-- margin 20 px-->
<div class="mrg-20"></div>
<!-- margin top 70 px-->
<div class="mrg-top-70"></div>
<!-- margin bottom 120 px-->
<div class="mrg-btm-120"></div>

paddings

use .pad-{10, 20, ... 120}, .pad-top-{10, 20, ... 120}, .pad-btm-{10, 20, ... 120} classes


<!-- padding 20 px-->
<div class="pad-20"></div>
<!-- padding top 70 px-->
<div class="pad-top-70"></div>
<!-- padding bottom 120 px-->
<div class="pad-btm-120"></div>

separators

Use class separator, separator-left or separator-right


<!-- center separator -->
<p class="separator"></p>
<!-- left separator -->
<p class="separator-left"></p>
<!-- right separator -->
<p class="separator-right"></p>

parallax

For Parallax Backgrounds use data-stellar-background-ratio=""(background scroll speed) attribute


<section style="background-image: url('image.jpg')" data-stellar-background-ratio="0.2">
	...
</section>

overlays


<!-- LIGHT OVERLAY -->
<div class="overlay overlay-light"></div>

<!-- DARK OVERLAY -->
<div class="overlay overlay-dark"></div>

To make overlay darker or brighter use -2x, -3x, -4x suffixes


<div class="overlay overlay-dark-2x"></div>
<div class="overlay overlay-dark-3x"></div>
<div class="overlay overlay-dark-4x"></div>
<div class="overlay overlay-light-2x"></div>
<div class="overlay overlay-light-3x"></div>
<div class="overlay overlay-light-4x"></div>

Example:


<section style="background-image: url('image.jpg')">

	<div class="overlay overlay-dark-2x"></div>

	<div class="text-light">
		...
	</div>

</section>

font icons

pixeden font icons

You can place Pixeden Font Icons using the CSS Prefix pe-7s- and the icon's name.


<i class="pe-7s-photo"></i>

fontAwesome

CSS Prefix fa and the icon's name.


<i class="fa fa-photo"></i>

responsive videos

Place Youtube or Vimeo videos inside .video-container element


<div class="video-container">
	<iframe src="http://player.vimeo.com/video/26757344?title=0&byline=0&portrait=0"></iframe>
</div>

youtube backgrounds

add ytbg class and data-property attribute, Set videoURL


<div class="ytbg" data-property="{videoURL:'https://www.youtube.com/watch?v=I5sJhSNUkwQ',containment:'self'}">
	...
</div>

<a href="#portfolio" class="sscroll button"></a>
